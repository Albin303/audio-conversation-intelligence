## 💡 EXAMPLE USAGE

### Via Browser (Easiest)
1. Go to http://localhost:5173
2. Paste text or upload audio
3. Click "Analyze"
4. See results instantly

### Via API (For Developers)
```bash
curl -X POST "http://localhost:8000/api/analyze" \
  -F "text=The product is great but expensive"
```

### Response Example
```json
{
  "products": [
    {"name": "product", "sentiment": "positive", "score": 0.78},
    {"name": "shipping", "sentiment": "positive", "score": 0.65}
  ],
  "summary": {
    "positive": 100,
    "negative": 0,
    "neutral": 0,
    "averageScore": 0.715
  }
}
```
